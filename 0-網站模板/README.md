複製整個資料夾的語法

* [xcopy - 檔案. 資料夾複製指令 @ 藍影 :: 痞客邦 ::](http://edisonshih.pixnet.net/blog/post/28109815-xcopy---%E6%AA%94%E6%A1%88.%E8%B3%87%E6%96%99%E5%A4%BE%E8%A4%87%E8%A3%BD%E6%8C%87%E4%BB%A4)
* 參數S，代表連當前目錄底下的資料夾也複製
* 參數改成E，就是空目錄也會複製

```bash
xcopy "A資料夾路徑" "B資料夾路徑" /S
```
